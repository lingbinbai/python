### 单选结构
```
if True :
    print("你好python.")
```


### 双选结构
```
if True :
    print("你好pyton.")
else :
    print("再见pyton.")
```

### 多选结构
```
number = 1
if number < 0 :
    print("number < 0")
elif number > 0 :
    print("number > 0")
else :
    print("number = 0")

```

### if嵌套
```
number = 1
if True :
    if number < 0 :
        print("number < 0")
    elif number > 0 :
        print("number > 0")
    else :
        print("number = 0")
```
